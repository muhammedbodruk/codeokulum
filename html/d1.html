HTML Ders 1: HTML'in Tarihi ve Temel Bilgiler

Bu derste, HTML'in tarihçesi ve temel bilgiler üzerinde duracağız.

HTML'in Tarihi

HTML (HyperText Markup Language), 1991 yılında Tim Berners-Lee tarafından geliştirilmiştir. İlk web sayfası bu dil kullanılarak yazılmıştır.

HTML Temelleri

HTML, web sayfalarının yapısını oluşturmak için kullanılan işaretleme dilidir. HTML etiketleri ile metin, görsel, bağlantı gibi elemanlar tanımlanır. 

<!DOCTYPE html>
<html>
  <head>
    <title>Başlık</title>
  </head>
  <body>
    <h1>Merhaba Dünya</h1>
  </body>
</html>
