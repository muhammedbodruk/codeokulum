Java Ders 1: Java'ya Giriş ve Temel Bilgiler

Bu derste, Java programlama dilinin tarihçesi ve temel özellikleri hakkında bilgi edineceksiniz.

Java'nın Tarihi

Java, Sun Microsystems tarafından 1995 yılında geliştirildi. James Gosling tarafından yaratılan bu dil, "Write Once, Run Anywhere" felsefesi ile tanınır.

Java'nın Temel Özellikleri

  • Platform bağımsızlığı
  • Güvenlik
  • Çok iş parçacıklı programlama
  • Dinamik bellek yönetimi
