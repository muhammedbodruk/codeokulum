JavaScript Ders 7: Obje Yapısı ve JSON

Bu derste, JavaScript'te obje yapısını ve JSON formatını öğreneceksiniz.

JavaScript Objeleri

Objeler, özellik ve değer çiftlerini tutan veri yapılarıdır: 

let kisi = {
  ad: "Ahmet",
  yas: 30,
  sehir: "İstanbul"
};

JSON (JavaScript Object Notation)

JSON, veri değişimi için hafif bir formattır:

let jsonStr = '{"ad": "Ahmet", "yas": 30, "sehir": "İstanbul"}';
let obj = JSON.parse(jsonStr);
console.log(obj.ad); // Ahmet
