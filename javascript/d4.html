JavaScript Ders 4: Operatörler ve Koşullu İfadeler

Bu derste, JavaScript'teki operatörler ve koşullu ifadeler hakkında bilgi edineceksiniz.

Operatörler

JavaScript'te çeşitli operatörler vardır:

  • Atama Operatörleri: =, +=, -=, *=, /=
  • Aritmetik Operatörler: +, -, *, /, %
  • Karşılaştırma Operatörleri: ==, ===, !=, !==, >, <, >=, <=
  • Mantıksal Operatörler: && (ve), || (veya), ! (değil)

Koşullu İfadeler

JavaScript'te koşullu ifadeler şunlardır:

  • if ifadesi:
    • if (kosul) {
  // kod bloğu
}
  • else ifadesi:
    • if (kosul) {
  // kod bloğu
} else {
  // başka kod bloğu
}
  • else if ifadesi:
    • if (kosul1) {
  // kod bloğu
} else if (kosul2) {
  // başka kod bloğu
} else {
  // başka kod bloğu
}
  • switch ifadesi:
    • switch (deger) {
  case deger1:
    // kod bloğu
    break;
  case deger2:
    // başka kod bloğu
    break;
  default:
    // başka kod bloğu
}
