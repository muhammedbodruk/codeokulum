JavaScript Ders 3: Değişkenler ve Veri Türleri

Bu derste, JavaScript'te değişkenlerin nasıl tanımlandığını ve farklı veri türlerini öğreneceksiniz.

Değişken Tanımlama

JavaScript'te değişkenler var, let veya const anahtar kelimeleri ile tanımlanır. 

var isim = 'Ali';
let yas = 25;
const dogumYili = 1999;

Veri Türleri

JavaScript'te yaygın olarak kullanılan veri türleri şunlardır:

  • String: Metin verilerini temsil eder. Örnek: 'Merhaba'
  • Number: Sayısal verileri temsil eder. Örnek: 42
  • Boolean: Doğru veya yanlış değerlerini temsil eder. Örnek: true veya false
  • Array: Liste verilerini temsil eder. Örnek: [1, 2, 3]
  • Object: Anahtar-değer çiftlerini temsil eder. Örnek: { isim: 'Ali', yas: 25 }
  • Undefined: Tanımlanmamış değişkenleri temsil eder. Örnek: let tanimsiz;
  • Null: Boş veya geçersiz değeri temsil eder. Örnek: null

Örnek Kullanım

Bir değişkeni tanımlayıp veri türünü değiştirmek:

let degisken = 'Merhaba';  // String türünde
degisken = 42;           // Number türünde
