JavaScript Ders 6: Diziler ve Dizi İşlemleri

Bu derste, JavaScript'te diziler ve dizi işlemleri hakkında bilgi edineceksiniz.

JavaScript Diziler

Diziler, birden fazla veriyi tek bir değişken altında saklamamıza olanak tanır. JavaScript'te diziler şu şekilde tanımlanır: 

let meyveler = ["Elma", "Muz", "Çilek"];

Dizi Metotları ve İşlevleri

JavaScript dizileri üzerinde çeşitli işlemler yapmamıza olanak tanıyan birçok yerleşik metod sağlar. İşte bazı yaygın kullanılan dizi metodları:

  • push(): Dizinin sonuna bir veya daha fazla eleman ekler.
  • pop(): Dizinin son elemanını kaldırır.
  • shift(): Dizinin ilk elemanını kaldırır.
  • unshift(): Dizinin başına bir veya daha fazla eleman ekler.
  • splice(): Belirtilen pozisyonda eleman ekler veya kaldırır.
  • slice(): Belirtilen pozisyondaki elemanları yeni bir dizi olarak döner.

Dizilerle İşlemler

Dizilerle döngüler kullanarak çeşitli işlemler yapabiliriz. Örneğin: 

// Diziyi yazdırmak için for döngüsü
let meyveler = ["Elma", "Muz", "Çilek"];
for (let i = 0; i < meyveler.length; i++) {
  console.log(meyveler[i]);
}

// forEach metodu ile dizi üzerinde gezinmek
meyveler.forEach(function(meyve) {
  console.log(meyve);
});
