JavaScript Ders 5: Döngüler ve Fonksiyonlar

Bu derste, JavaScript'te döngüler ve fonksiyonlar hakkında bilgi edineceksiniz.

Döngüler

JavaScript'te döngüler, belirli bir kod bloğunu birden çok kez çalıştırmak için kullanılır. Yaygın döngüler:

  • for döngüsü:
    • for (let i = 0; i < 10; i++) {
  console.log(i);
}
  • while döngüsü:
    • let i = 0;
while (i < 10) {
  console.log(i);
  i++;
}
  • do...while döngüsü:
    • let i = 0;
do {
  console.log(i);
  i++;
} while (i < 10);
  • for...in döngüsü:
    • const obj = {a: 1, b: 2, c: 3};
for (let key in obj) {
  console.log(`${key}: ${obj[key]}`);
}
  • for...of döngüsü:
    • const arr = [1, 2, 3];
for (let value of arr) {
  console.log(value);
}

Fonksiyonlar

JavaScript'te fonksiyonlar, belirli bir görevi gerçekleştirmek için kullanılan kod bloklarıdır. Fonksiyon tanımlama ve çağırma örnekleri:

  • Fonksiyon Bildirimi:
    • function merhaba() {
  console.log('Merhaba Dünya');
}
merhaba();
  • Fonksiyon İfadesi:
    • const merhaba = function() {
  console.log('Merhaba Dünya');
};
merhaba();
  • Ok Fonksiyonları:
    • const merhaba = () => {
  console.log('Merhaba Dünya');
};
merhaba();
