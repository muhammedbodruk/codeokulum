CSS Ders 2: CSS Seçiciler ve Özellikler

Bu derste, CSS seçicileri ve özellikleri hakkında bilgi sahibi olacaksınız.

CSS Seçiciler

CSS seçicileri, HTML elemanlarını seçmek için kullanılır. İşte bazı temel seçiciler:

Element Seçiciler

p {
  color: red;
}

Sınıf Seçiciler

.sari {
  color: yellow;
}

ID Seçiciler

#kirmizi {
  color: red;
}

CSS Özellikleri

CSS, HTML elemanlarının stilini değiştirmek için birçok özellik sunar. İşte bazı temel özellikler:

Renk Özellikleri

color: blue;
background-color: yellow;

Yazı Tipi Özellikleri

font-size: 16px;
font-family: Arial, sans-serif;

Kutu Modeli Özellikleri

margin: 10px;
padding: 10px;
border: 1px solid black;
