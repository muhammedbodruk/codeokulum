CSS Ders 5: CSS Animasyonlar ve Geçişler

Bu derste, CSS animasyonları ve geçişleri kullanmayı öğreneceksiniz.

CSS Geçişler

Geçişler, CSS özelliklerinin belirli bir süre içinde nasıl değişeceğini kontrol eder. 

div {
  transition: width 2s;
}
div:hover {
  width: 200px;
}

CSS Animasyonlar

Animasyonlar, CSS ile elemanlara hareket ve efekt eklemenin bir yoludur.

Keyframes

@keyframes example {
  from {background-color: red;}
  to {background-color: yellow;}
}
div {
  animation-name: example;
  animation-duration: 4s;
}
