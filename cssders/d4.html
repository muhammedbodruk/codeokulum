CSS Ders 4: CSS Flexbox ve Grid Layout

Bu derste, CSS Flexbox ve Grid Layout'unu öğreneceksiniz.

CSS Flexbox

Flexbox, sayfa düzenlerini oluşturmak için kullanılan güçlü bir CSS modelidir.

Flex Container

div.container {
  display: flex;
}

Flex Items

div.container > div {
  flex: 1;
}

CSS Grid Layout

Grid Layout, ızgara tabanlı bir düzen oluşturmak için kullanılır.

Grid Container

div.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}

Grid Items

div.container > div {
  grid-column: span 2;
}
