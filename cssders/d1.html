CSS Ders 1: CSS'in Temel Bilgileri ve Kullanımı

Bu derste, CSS'in temel bilgileri ve kullanımını öğreneceksiniz.

CSS Temel Bilgileri

CSS (Cascading Style Sheets), HTML belgelerinin görünümünü biçimlendirmek için kullanılan bir dildir. 

body {
  background-color: lightblue;
}

CSS'in HTML'e Eklenmesi

CSS, HTML'e üç şekilde eklenebilir: inline, internal ve external.

Inline CSS

<p style="color:blue;">Bu mavi renkte bir paragraf.</p>

Internal CSS

<style>
p {
  color: blue;
}
</style>

External CSS

<link rel="stylesheet" href="styles.css">
