Python Ders 4: Python Operatörler ve Koşullu İfadeler

Bu derste, Python'da operatörler ve koşullu ifadeler hakkında bilgi edineceksiniz.

Python Operatörler

Python'da çeşitli operatörler kullanılır:

  • Aritmetik Operatörler: +, -, *, /, %
  • Karşılaştırma Operatörleri: ==, !=, <, >, <=, >=
  • Mantıksal Operatörler: and, or, not 
# Örnek Kullanım
a = 5
b = 3
print(a + b)  # 8
print(a > b)  # True
print(a == b)  # False
print(a != b and a > b)  # True

Python Koşullu İfadeler

Python'da koşullu ifadeler if, elif ve else ile oluşturulur: 

# Koşullu İfadeler Örneği
x = 10
if x > 0:
    print("Pozitif")
elif x == 0:
    print("Sıfır")
else:
    print("Negatif")
