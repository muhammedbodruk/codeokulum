Python Ders 2: Python Kurulumu ve Çalıştırma

Bu derste, Python'un nasıl kurulacağını ve çalıştırılacağını öğreneceksiniz.

Python Kurulumu

Python'u python.org adresinden indirebilirsiniz.

  • Windows: İndirme sayfasından Python'u indirip kurun.
  • Mac: Python'u indirip kurun veya brew install python komutunu kullanın.
  • Linux: Paket yöneticisi ile Python'u kurun. Örneğin, Debian tabanlı dağıtımlarda sudo apt-get install python3.

Python Çalıştırma

Python'u kurduktan sonra, terminal veya komut istemcisinden python veya python3 komutlarıyla çalıştırabilirsiniz. 

# Terminal veya komut istemcisinde:
python --version
python3 --version
