Python Ders 3: Python Değişkenler ve Veri Türleri

Bu derste, Python'da değişkenler ve veri türleri hakkında bilgi edineceksiniz.

Python Değişkenler

Değişkenler, verileri saklamak için kullanılır. Python'da değişkenler herhangi bir türde veri saklayabilir. 

# Değişken tanımlama
x = 5
y = "Merhaba, Dünya!"
z = 3.14

Python Veri Türleri

Python'da çeşitli veri türleri vardır, bunlar arasında tamsayılar, kayan noktalı sayılar, metinler ve listeler bulunur.

  • int: Tamsayılar
  • float: Kayan noktalı sayılar
  • str: Metinler
  • list: Listeler
