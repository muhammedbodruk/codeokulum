Python Ders 5: Python Döngüler ve Fonksiyonlar

Bu derste, Python'da döngüler ve fonksiyonlar hakkında bilgi edineceksiniz.

Python Döngüler

Python'da döngüler, kod bloklarını belirli koşullar altında tekrar tekrar çalıştırmak için kullanılır:

  • for döngüsü
  • while döngüsü
# for döngüsü örneği
for i in range(5):
    print(i)  # 0'dan 4'e kadar olan sayıları yazdırır

# while döngüsü örneği
i = 0
while i < 5:
    print(i)
    i += 1

Python Fonksiyonlar

Fonksiyonlar, belirli bir görevi yerine getiren kod bloklarıdır. def anahtar kelimesi ile tanımlanır: 

# Fonksiyon tanımlama ve çağırma
def merhaba():
    print("Merhaba, Dünya!")

merhaba()  # Fonksiyonu çağırır

Fonksiyonlar ayrıca parametre alabilir ve değer döndürebilir: 

# Parametreli fonksiyon ve dönüş değeri
def topla(a, b):
    return a + b

sonuc = topla(3, 5)
print(sonuc)  # 8
